# Components

`@kreftforeningen/web-react` delivers the production-ready UI we use across apps: buttons, forms, dialogs, navigation, and more. Each story here demonstrates the defaults, recommended props, and accessibility affordances that ship with the package.

- [NPM](https://www.npmjs.com/package/@kreftforeningen/web-react)
- [GitHub](https://github.com/kreftforeningen/web-react)

## Install

```bash
pnpm add @kreftforeningen/web-react
```

## Usage

```tsx
import { Button } from "@kreftforeningen/web-react";

export const SaveAction = () => (
  <Button variant="primary" onClick={handleSave}>
    Save changes
  </Button>
);
```
