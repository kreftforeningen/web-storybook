# @kreftforeningen/web-css

Package to be used as a base for @kreftforeningen/web-react.

- [NPM.js](https://www.npmjs.com/package/@kreftforeningen/web-css)
- [GitHub](https://github.com/kreftforeningen/web-css/)

## Install

```
pnpm add @kreftforeningen/web-css
```

## Usage

Import it at the top of your main css file, like this:

```
@import "@kreftforeningen/web-css";
```
