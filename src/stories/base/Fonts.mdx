# Fonts

Our design system uses IBM Plex font family ([SIL Open Font License 1.1](https://github.com/IBM/plex/blob/master/LICENSE.txt)) with three main variants:

- **Sans**: IBM Plex Sans - The primary font for body text and general content
- **Condensed**: IBM Plex Condensed - Used for headings and titles where space is limited
- **Serif**: IBM Plex Serif - Used for blockquotes and special content that needs emphasis

## Download

- Fonts are included in `@kreftforeningen/web-css`, read below for usage
- Also available to download at google-webfonts-helper: [sans](https://gwfh.mranftl.com/fonts/ibm-plex-sans?subsets=latin), [condensed](https://gwfh.mranftl.com/fonts/ibm-plex-sans-condensed?subsets=latin) and [serif](https://gwfh.mranftl.com/fonts/ibm-plex-serif?subsets=latin),

## Usage

Each font variant is available as a variable:

- `font-sans` - Default body text
- `font-condensed` - Condensed headings
- `font-serif` - Serif text for quotes
